# AI 游戏交互 Demo

一个面向横屏体验设计的网页 Demo：使用本地 MP4 作为游戏画面，完整展示视频内容并允许黑边/背景，同时在底部提供 AI 提示词输入与发送交互。视频会在页面加载 4 秒后开始静音循环播放。

## 运行

```bash
npm install
npm run start
```

然后在浏览器打开终端提示的地址。电脑和移动设备预览时页面会以横屏舞台居中显示，视频使用完整展示布局，不会裁切画面。

## 文件说明

- `index.html`：页面结构
- `styles.css`：横屏舞台、视频完整展示与底部输入区视觉样式
- `app.js`：视频延迟播放、输入框发送、回车发送、自动高度、模拟 AI 回复
- `assets/game-video.mp4`：网页中引用的视频资源
